# Security policy

Please report vulnerabilities privately through GitHub Security Advisories for `kdnsna/decklint`.

DeckLint treats PPTX files as untrusted ZIP/XML input. Reports should include the DeckLint version, operating system, the smallest safe reproduction, and whether LibreOffice rendering was enabled. Do not attach confidential presentations to public issues.

The project supports the latest released minor version. DeckLint never resolves external relationships and v0.1 never modifies the source presentation.
