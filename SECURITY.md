# Security policy

Please report vulnerabilities privately through GitHub Security Advisories for `kdnsna/pptlint`.

PPTLint treats PPTX files as untrusted ZIP/XML input. Reports should include the PPTLint version, operating system, the smallest safe reproduction, and whether LibreOffice rendering was enabled. Do not attach confidential presentations to public issues.

The project supports the latest released minor version. PPTLint never resolves external relationships and never modifies the source presentation.
